Who?

My name is Andrew Jonhardt, and I'm both an indie and Freelance game developer. My skills lie in:

  • The Godot game engine

    I've been using Godot on and off since 2019 as a hobbyist. I failed to get very far just focusing on my own projects, and I've resolved to turn my focus towards the needs of others.

  • Game Design

    I've been studying games and game design since 2018, when I first decided to try creating a game with the assistance of ATX Game Designers. I've learned what speaks to me most as a designer (area control + tactics and strategy), what a game is (check out

    Game Design Theory
    by Keith Burgun), how to give and accept critism, and how to push a game from idea to completion.
